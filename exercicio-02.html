Exercício: Integração com Busca do Google

Instruções Pedagógicas

  • Desafio: Criar um formulário HTML capaz de enviar uma consulta diretamente para o sistema de buscas do Google.
  • Conceitos Chave: Atributos fundamentais da tag <form> e parâmetros de URL (Query String).
  • Requisitos Técnicos obrigatórios:
    • O atributo action deve apontar para a URL correta do script de busca do Google.
    • O atributo method deve usar o método HTTP apropriado para buscas.
    • O campo de texto de entrada precisa ter o atributo name com o valor exato exigido pela API pública do Google.
